Integration with TinyMCE

By default TinyMCE don't have source code highlighting functionality. This is commercial feature. But the most of web sites with installed TinyMCE don't have it. Our extension allows to use source code highlighting on absolutely any web site and adds a number of cool features like formatting and etc.

Integration with Summernote

Summernote have an ability to enable CoreMirror support out-of-the-box. Unfortunately, not all the projects which use Summernote adding it's support by default. So, that is why we've been requested to add Summernote support in the extension.

Editor Settings
